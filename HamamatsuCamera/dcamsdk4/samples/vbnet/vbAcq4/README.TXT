vbAcq4 is VB.NET sample code to get the image with DCAM-API.

The following files are main files.
dcamapi4.vb
	loads DCAM-API library and defines classes (structures) to control DCAM-API.
MyDcam.vb
	defines classes for easy use of dcamapi4.vb.
subacq4.vb
	defines class for processing to display images.
FormInfo.vb
	Form to show information of the device and DCAM-API
FormMain
	main Form to control DCAM-API function and display image.
FormProperties.vb
	Form to control DCAM-API properties.
